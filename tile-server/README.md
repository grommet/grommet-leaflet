# Self-hosted tile server

This is a self-hosted tile server for [OpenStreetMap](https://www.openstreetmap.org/) data. It is based on [OpenMapTiles](https://openmaptiles.org/) and [TileServer GL](https://tileserver.readthedocs.io/en/latest/).

## Usage

### Prerequisites

- Node
- Can be run on Docker

.mbtiles file is not included. Need to download separately.
